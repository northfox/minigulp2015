# minigulp

gulp environment for simple services

## Tasks

### Preparing

    npm install
    bower install

### Begin to watching files

    gulp

### Minify for release

    gulp -min

## Dependency services

- [Gulp](http://gulpjs.com/)
- [WebPack](http://webpack.github.io/)
- [Bower](http://bower.io/)
